# Number Guessing Game

A simple multiplayer number guessing game built with Python.

## Description

This is an interactive game where up to 3 players compete to guess random numbers. The game runs for 5 rounds, and the player with the highest score at the end wins

## How to Play

1. Run the game: `python game.py`
2. Each round, the game generates a random number
3. Each player takes turns guessing the number
4. If you guess correctly, you earn a point
5. After 5 rounds, final scores are displayed

## Requirements

- Python 3.x

## How to Run

```bash
python game.py
```

## Players

The game supports 3 players

## Scoring

- Correct guess = 1 point
- Incorrect guess = 0 points
- Winner is determined by the highest total score after 5 rounds
